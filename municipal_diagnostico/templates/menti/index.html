App Mentimeter standalone

La app con rutas literales /admin y /join se mantiene como servicio independiente para no interferir con el /admin del sistema municipal. Configura MENTI_PUBLIC_URL en produccion para redirigir esta entrada al servicio publicado.

Volver a Live

Comando local

cd mentimeter_live_app
python -m pip install -r requirements.txt
python app.py

Después abre http://127.0.0.1:5000/admin y usa el demo 123456.
